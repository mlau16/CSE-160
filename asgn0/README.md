# Assignment 0: Vector Library

## Overview

Uses the matrix library provided by the textbook "WebGL Programming Guide" by Matsuda & Lea to create a simple program that draws 2 vectors and performs operations on them. This assignment extends the textbook's cuon-matrix library to support a full set of vector operations and visualizes them on an HTML <canvas> element. 

---
## How To Use
1. Open asg0.html in web browser (or use github pages)
2. Enter values in input boxes
3. Click Draw
4. View numerical results in browser console.
